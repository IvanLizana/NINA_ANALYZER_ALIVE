# NINA_ANALYZER_ALIVE
Precision metrology tool for N.I.N.A. astrophotography logs. Features real-time monitoring, true RMS guiding calculations, and forensic PHD2 star diagnosis (Heatmaps/Bullseye). Includes session efficiency KPIs and a 'Deep Dark' UI to preserve night vision. Portable, standalone app powered by Python &amp; Streamlit.
